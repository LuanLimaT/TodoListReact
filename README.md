Lista de Tarefas (Todo App)

Um aplicativo simples de lista de tarefas desenvolvido com React.

📌 Funcionalidades

Adicionar novas tarefas com categoria

Pesquisar tarefas

Marcar tarefas como concluídas

Remover tarefas

Listagem ordenada em ordem alfabética

React

JavaScript
